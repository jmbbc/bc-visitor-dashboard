ASSALAMUALAIKUM WARAHMATULLAHI WABARAKATUH

Pihak Jawatankuasa Pengurusan Bersama (JMB) dengan kerjasama Surau Al-Muhtadin, Banjaria Court merancang untuk menganjurkan satu program cuti sekolah khas untuk anak-anak komuniti Banjaria Court.

Program ini merupakan Program Cuti Sekolah Keluarga Besar Kami selama 3 hari, dikendalikan oleh Alfadir Ustaz Meor Abdullah. Program ini bertujuan mengisi masa lapang remaja dengan pengisian bermanfaat merangkumi kerohanian, sahsiah dan pembentukan jati diri.
